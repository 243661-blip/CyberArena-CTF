 # CyberArena

CyberArena is a modern Capture The Flag (CTF) platform designed for cybersecurity competitions, ethical hacking practice, and security education.

## Features

- Dynamic challenge system
- Team-based competitions
- Scoreboard and rankings
- Secure user authentication
- Plugin and theme support
- Docker deployment support
- Challenge categories and hints
- Admin dashboard
- Security-focused architecture

## Installation

1. Install dependencies:
   pip install -r requirements.txt

2. Start the platform:
   python serve.py

3. Open browser:
   http://localhost:8000
